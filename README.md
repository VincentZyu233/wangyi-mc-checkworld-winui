# 网易MC检查世界

WinUI3 桌面版 | Version 0.1.18

## 功能
- 📂 列出所有网易 MC 存档
- 🔍 搜索和排序
- 💾 备份存档为 ZIP
- ✏️ 重命名存档
- 🗑️ 删除存档

## 运行
双击 exe 文件即可运行（需要 Windows 10/11 x64）

## 开发
```powershell
dotnet build
dotnet run
```
